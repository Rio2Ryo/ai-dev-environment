# Manusへの指示テンプレート

このドキュメントには、Manusに開発を依頼する際の指示テンプレートが含まれています。
コピー＆ペーストして使用してください。

## 🚀 基本テンプレート

### 新規開発

```
ai-dev-environmentをクローンして、以下の開発をClaude Codeに委譲してください。

【タスク】
[ここにタスクの詳細を記載]

【要件】
- [要件1]
- [要件2]
- [要件3]

【技術スタック】
- [使用する技術]

Claude Codeが完了したら、ブラウザで動作確認し、品質に問題があれば修正を指示してください。
最終的に満足できる品質になったら報告してください。
```

### バグ修正

```
ai-dev-environmentをクローンして、以下のバグ修正をClaude Codeに委譲してください。

【問題】
[バグの詳細な説明]

【再現手順】
1. [手順1]
2. [手順2]
3. [手順3]

【期待する動作】
[正しい動作の説明]

修正後、ブラウザで動作確認し、問題が解決したことを確認してください。
```

### 機能追加

```
ai-dev-environmentをクローンして、以下の機能追加をClaude Codeに委譲してください。

【追加する機能】
[機能の説明]

【既存プロジェクト】
[プロジェクトの場所やリポジトリURL]

【要件】
- [要件1]
- [要件2]

Claude Codeが完了したら、ブラウザで新機能を確認し、既存機能に影響がないことも確認してください。
```

## 📱 フロントエンド開発

### Reactコンポーネント作成

```
ai-dev-environmentをクローンして、以下のReactコンポーネント作成をClaude Codeに委譲してください。
frontend-designとreact-best-practicesスキルを使用してください。

【コンポーネント】
[コンポーネント名と説明]

【要件】
- [要件1]
- [要件2]

【デザイン】
- [デザインの詳細]

Claude Codeが完了したら、ブラウザで以下を確認してください：
- UIが正しく表示されるか
- レスポンシブ対応
- インタラクションが動作するか

問題があれば修正を指示し、品質が満足できるまで繰り返してください。
```

### ランディングページ作成

```
ai-dev-environmentをクローンして、以下のランディングページ作成をClaude Codeに委譲してください。
frontend-designスキルを使用してください。

【概要】
[サービス/プロダクトの説明]

【必要なセクション】
- ヒーローセクション
- 機能紹介
- 料金プラン
- FAQ
- フッター

【デザイン要件】
- モダンでクリーンなデザイン
- レスポンシブ対応
- ダークモード対応

Claude Codeが完了したら、ブラウザで全セクションを確認し、デザインの品質をチェックしてください。
```

## 💡 使い方のコツ

### 1. 具体的に指示する

❌ 悪い例：「いい感じのアプリを作って」

✅ 良い例：「React + TypeScriptでTodoアプリを作成してください。機能は追加・削除・完了切り替え。Tailwind CSSでモダンなデザインにしてください。」

### 2. スキルを指定する

フロントエンド開発の場合：
```
frontend-designとreact-best-practicesスキルを使用してください。
```

### 3. 品質確認を依頼する

```
Claude Codeが完了したら、ブラウザで動作確認し、品質に問題があれば修正を指示してください。
```

### 4. 繰り返しを許容する

```
最終的に満足できる品質になるまで、確認と修正を繰り返してください。
```
